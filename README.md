# Citrus Theme

Citrus Theme is an Obsidian theme inspired by the **MacBook Neo Citrus** color scheme.

It blends warm citrus yellows and greens with high-contrast typography to create a focused workspace that still feels vibrant.

## Highlights

- Balanced light and dark palettes with consistent contrast
- Readable headings with citrus-accented hierarchy
- Styled callouts, tags, tasks, tables, and code blocks
- OS-adaptive interface/text fonts inspired by Cupertino theme conventions
- Monospace-friendly code rendering for notes and snippets

## Inspiration

This theme draws from the MacBook Neo Citrus visual style: lively citrus tones, earthy depth, and clean readability for long writing sessions.

## Installation

### From Obsidian (after publication)

1. Open **Settings -> Appearance -> Themes**
2. Click **Manage**
3. Search for **Citrus Theme**
4. Install and enable it

### Manual install

1. Open your vault folder
2. Go to `.obsidian/themes/`
3. Create a folder named `Citrus Theme`
4. Place `manifest.json` and `theme.css` inside it
5. Reload Obsidian and select **Citrus Theme**

## Publishing checklist

- `manifest.json` is present and metadata is filled in
- `theme.css` is at repository root
- Add a screenshot thumbnail (recommended file name: `screenshot.png`)
  - Recommended aspect ratio: `16:9`
  - Recommended size: `512x288`
- Optional but recommended: add `versions.json` for release compatibility mapping

## Theme Gallery submission

1. Create a public GitHub repository for your theme
2. Add `manifest.json`, `theme.css`, `README.md`, and screenshot thumbnail
3. Submit a PR to `obsidianmd/obsidian-releases` to add your theme to Community Themes
4. For each GitHub release, upload `manifest.json` and `theme.css`

## Recommended setup

- Use default Obsidian accent settings for intended colors
- Keep your editor line length comfortable (readability-first layout)
- Pair with your preferred monospace coding font in Obsidian if needed

## Known notes

- Font behavior is platform-specific and follows system availability
- Some UI differences may appear across desktop and mobile builds
